# SSNnote (서서니 메모)

Windows 전용 경량 스티커 메모 앱. 고등학교 행정 업무 및 수학 연구(유자-pi) 관련 빠른 메모 작성에 특화.

## 코딩 규칙

- DB 로직(`db.py`), 창 UI(`window.py`), 진입점(`main.py`), 자동시작(`autostart.py`) 역할 분리 유지.
- UI는 깔끔하고 방해 요소 없이 유지.
- 새 기능 추가 시 DB 마이그레이션은 `init_db()`에 `ALTER TABLE` 형식으로 추가.

## 향후 로드맵

- 온라인 동기화 모드(로그인 시스템)로 전환 예정. 현재는 로컬 SQLite 성능에 집중.
